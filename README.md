# ✈️ Airport-Analysis-Dashboard

# Project Overview
This project analyzes airport operations including flights, passengers, delays, and airline performance using Power BI.

# Tools Used
- Power BI
- Excel

# Key Insights
- Identified busiest routes and peak travel periods
- Analyzed flight delays and cancellations
- Compared domestic vs international traffic
- Evaluated airline performance
